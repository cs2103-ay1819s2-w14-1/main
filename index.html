BrainTrain

  • This is a desktop flashcard application. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).

  • It is an open-source spaced repetition flashcard application which makes memorizing easy and efficient.

  • Research has shown that the Spaced-Repetition System (SRS) is more effective than traditional memorization techniques
